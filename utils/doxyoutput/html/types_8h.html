Mangles the name of a function with given argument types. Note that the return value is malloc'd, but after syntax tree gen, the original name is not needed. Mangling example: fun name(ptr string x, int a[266], int y, fun (int) -> void) -> string turns into: name_psi['10''3']fi->v where 'a' is the character with ascii code a. The characters are computed by converting the size to base 128, adding 1 to each character (to prevent null-termination issues), and ordering bytes in big-endian order
